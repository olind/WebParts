Robust way to clear floats

  • (Very) inspired by page 106 in Handcrafted CSS by Dan Cederholm
  • Works in IE6/7
  • Add the class group to the outer div

CSS

  .group:after {
      content: ".";
      display: block;
      height: 0;
      clear: both;
      visibility: hidden;
    }
    * html .group { /* for IE6 */
      height: 1%;
    }
    *:first-child+html .group { /* for IE7 */
      min-height: 1px;
    }
